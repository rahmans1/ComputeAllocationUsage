How to Get Access

Compute Canada Services

  • Make a compute canada account
  • For GREX, you need to apply for a westgrid consortium access on your compute canada account
  • For access to Niagara, you will need to opt-in
  • Same applies for creating cloud project

Jefferson LAB services

  • Make a jlab account
  • You also automatically have access to jlab virtual machines

Logging in

Compute Canada Services

Linux Machines * Beluga ** ssh <username>@beluga.computecanada.ca -Y ** ssh <username>@beluga<1/2/3/4> -Y * GREX ** ssh <username>@grex.westgrid.ca -Y * Cedar, Graham, Niagara ** ssh <username>@<cedar/graham/niagara>.computecanada.ca -Y * JLAB ** ssh <username>@login.jlab.org -Y ** ssh <username>@ifarm<1901/1902/1801/1802> -Y Windows Machines Use putty and xming

File Transfers

  • Preferably use globus
  • sftp from linux machines ** lcd <local directory> ** cd <remote directory> ** put -r * [ Transfers from local to remote ] ** get -r * [ Transfers from remote to local ]
  • psftp from windows machines
